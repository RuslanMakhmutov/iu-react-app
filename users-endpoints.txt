Пользователи

/users - GET - получение списка пользователей
200 - успешное получение пользователей
400 - неверный запрос
500 - ошибка сервера

/users - POST - добавление нового пользователя
200 - успешное добавление пользователя
400 - неверный запрос
401 - гостям доступ запрещён
403 - вам запрещено добавлять пользователя
500 - ошибка сервера

/users/{id} - GET - получение пользователя по ID
200 - успешное получение пользователя
400 - неверный запрос
500 - ошибка сервера

/users/{id} - PATCH - изменение пользователя по ID
200 - успешное изменение пользователя
400 - неверный запрос
401 - гостям доступ запрещён
403 - вам запрещено изменять пользователя
500 - ошибка сервера

/users/{id} - DELETE - удаление пользователя по ID
200 - успешное удаление пользователя
400 - неверный запрос
401 - гостям доступ запрещён
403 - вам запрещено удалять пользователя
500 - ошибка сервера
